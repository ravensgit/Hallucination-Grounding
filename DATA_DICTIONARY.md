# Data Dictionary: Output CSV Files

This document describes the structure and contents of CSV files generated by `milestone_3.py`.

---

## 1. coco_baseline_captions.csv

**Description:** Greedy BLIP captions for 5,000 COCO val2017 images (baseline method).

**Columns:**
- `image_id` (int): COCO image ID
- `fname` (str): Filename in COCO val2017 directory
- `caption` (str): Greedy BLIP caption (no sampling)

**Rows:** 5,000

**Example:**
```
image_id,fname,caption
391895,COCO_val2017_000000391895.jpg,"a man sitting on a bench in the park"
391896,COCO_val2017_000000391896.jpg,"a cat lying on a couch"
```

---

## 2. coco_sampled_captions.csv

**Description:** 5 nucleus-sampled alternative captions for each image.

**Columns:**
- `image_id` (int): COCO image ID
- `fname` (str): Filename
- `sample_1` (str): First nucleus-sampled caption (top_p=0.9)
- `sample_2` (str): Second nucleus-sampled caption
- `sample_3` (str): Third nucleus-sampled caption
- `sample_4` (str): Fourth nucleus-sampled caption
- `sample_5` (str): Fifth nucleus-sampled caption

**Rows:** 5,000

**Use:** These form the candidate pool for selection (pool = [greedy] + [5 samples])

---

## 3. coco_sample_variation.csv

**Description:** CHAIR scores for each sample position individually (to show variation).

**Columns:**
- `sample_position` (int): Which of the 5 samples (1–5)
- `CHAIRs` (float): Sentence-level hallucination rate (%)
- `CHAIRi` (float): Instance-level hallucination rate (%)
- `hall_caps` (int): Number of hallucinated captions
- `hall_objs` (int): Number of hallucinated object mentions

**Rows:** 5 (one per sample position)

**Example:**
```
sample_position,CHAIRs,CHAIRi,hall_caps,hall_objs
1,5.34,3.67,267,183
2,4.74,3.12,237,156
3,5.12,3.45,256,172
4,5.64,3.89,282,194
5,5.00,3.24,250,162
```

**Interpretation:** Sample variation ranges from 4.74% to 5.64%, confirming that none of the 5 individual positions beats the hybrid (0.74%). This proves the hybrid actively selects per-image, not exploiting one good sample position.

---

## 4. coco_baseline_chair.csv

**Description:** Detailed CHAIR analysis for baseline (greedy) captions.

**Columns:**
- `image_id` (int): COCO image ID
- `fname` (str): Filename
- `caption` (str): Generated caption
- `gt` (str): Ground-truth objects (pipe-separated: "dog|cat|table")
- `mentioned` (str): Objects mentioned in caption (pipe-separated)
- `wrong` (str): Hallucinated objects not in ground truth (pipe-separated)
- `is_hall` (bool): True if caption has ≥1 hallucinated object

**Rows:** 5,000

**Example:**
```
image_id,fname,caption,gt,mentioned,wrong,is_hall
391895,COCO_val2017_000000391895.jpg,"a man with a dog",dog|person|bench,man|dog,man,True
391896,COCO_val2017_000000391896.jpg,"a cat on couch",cat|couch,cat|couch,,False
```

---

## 5. coco_clip_chair.csv

**Description:** Detailed CHAIR analysis for CLIP-only selection captions.

**Same structure as coco_baseline_chair.csv** but for the CLIP-only selection method.

**Expected result:** CHAIRs ~5.10% (worse than baseline due to CLIP favoring longer, more descriptive captions).

---

## 6. coco_hybrid_chair.csv

**Description:** Detailed CHAIR analysis for hybrid selection captions (our method).

**Same structure as coco_baseline_chair.csv** but for the hybrid selection method.

**Expected result:** CHAIRs ~0.74% (82% relative reduction from baseline).

---

## 7. coco_ablation_results.csv

**Description:** Summary results for all 7 ablation configurations.

**Columns:**
- `method` (str): Configuration name
- `alpha` (float): CLIP weight (α)
- `beta` (float): Object penalty weight (β)
- `gamma` (float): Length penalty weight (γ)
- `CHAIRs` (float): Sentence-level hallucination rate (%)
- `CHAIRi` (float): Instance-level hallucination rate (%)
- `hall_caps` (int): Number of hallucinated captions
- `hall_objs` (int): Number of hallucinated object mentions
- `avg_length` (float): Average caption length in words
- `avg_objects` (float): Average number of objects mentioned per caption

**Rows:** 7 (one per configuration)

**Configurations:**
1. `baseline_greedy` — No selection (baseline)
2. `clip_only` — α=1.0, β=0.0, γ=0.0 (CLIP only, no penalties)
3. `length_penalty_only` — α=1.0, β=0.0, γ=0.2 (length penalty only)
4. `object_penalty_only` — α=1.0, β=0.3, γ=0.0 (object penalty only)
5. `hybrid_default` — α=1.0, β=0.3, γ=0.2 (our method)
6. `weak_penalty` — α=1.0, β=0.15, γ=0.1 (weaker penalties)
7. `strong_penalty` — α=1.0, β=0.5, γ=0.3 (stronger penalties)

**Example:**
```
method,alpha,beta,gamma,CHAIRs,CHAIRi,hall_caps,hall_objs,avg_length,avg_objects
baseline_greedy,-,-,-,4.08,2.63,204,131,9.69,1.59
clip_only,1.0,0.0,0.0,5.10,3.48,255,174,10.24,1.51
object_penalty_only,1.0,0.3,0.0,0.90,0.87,45,43,9.48,1.15
hybrid_default,1.0,0.3,0.2,0.74,0.74,37,37,9.25,1.02
strong_penalty,1.0,0.5,0.3,0.64,0.66,32,33,8.95,0.89
```

**Key insight:** Object penalty does ~90% of the work (0.90% vs 4.08% baseline). Diminishing returns beyond default (0.74% vs 0.64% for strong).

## Analysis Tips

### Finding Interesting Hallucinations

```python
import pandas as pd

# Load detailed results
baseline = pd.read_csv("coco_baseline_chair.csv")
hybrid = pd.read_csv("coco_hybrid_chair.csv")

# Find images where hybrid fixed baseline
baseline_hall = baseline[baseline['is_hall'] == True]
hybrid_clean = hybrid[hybrid['is_hall'] == False]
improved = baseline_hall[baseline_hall['image_id'].isin(hybrid_clean['image_id'])]

print(f"Images improved: {len(improved)}")
for _, row in improved.head(5).iterrows():
    print(f"\nImage {row['image_id']}:")
    print(f"  Baseline: {row['caption']}")
    print(f"  Wrong: {row['wrong']}")
```

### Compute Statistics

```python
# Load ablation results
ablation = pd.read_csv("coco_ablation_results.csv")

# Find best configuration
best = ablation.loc[ablation['CHAIRs'].idxmin()]
print(f"Best config: {best['method']} (CHAIRs={best['CHAIRs']}%)")

# Analyze object penalty contribution
object_only = ablation[ablation['method'] == 'object_penalty_only']['CHAIRs'].values[0]
clip_only = ablation[ablation['method'] == 'clip_only']['CHAIRs'].values[0]
baseline = ablation[ablation['method'] == 'baseline_greedy']['CHAIRs'].values[0]

object_contrib = (baseline - object_only) / (baseline - clip_only)
print(f"Object penalty accounts for {object_contrib*100:.1f}% of improvement")
```

```python
# wrong          object (str)

```

## Missing Values

- No missing values in any CSV (all images successfully processed)
- Empty strings in `wrong` column indicate no hallucinations for that caption
- Pipe-separated (`|`) strings can be split with `.str.split('|')` in pandas
